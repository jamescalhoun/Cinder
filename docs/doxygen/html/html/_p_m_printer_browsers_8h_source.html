111 /*
112 Printer browser module-specific keys for the printer lookup specification
113 dictionary. This CFDictionary is supplied by the printer module and provides
114 the printer browser module with the information it needs to browse for a given
115 type of printer over its connection type, and to display it in the browser
116 view.
117 
118 The printer module supplies the following keys and values (these are more
119 fully defined in PMPrinterModule.h):
120 
121 kPMPrBrowserLookupKindKey: a CFStringRef to a human-readable printer type
122 string that can be displayed in the browser view.
123 
124 kPMPrBrowserLookupInfoKey: a CFPropertyListRef to connection-specific
125 information used to look up a particular kind of printer over a given IO
126 connection.
127 
128 kPMPrBrowserLookupIconKey: a CFDataRef to the icon family data for the
129 printer icon to be displayed in the browser view.
130 
131 In addition, PrintCenter adds the following information to the lookup
132 specification dictionary:
133 
134 kPMPrBrowserLookupRefKey: a CFTypeRef to PrintCenter-specific information
135 that must be returned by the printer browser module in the printer selection
136 specfication dictionary (see below) when a printer is selected that was
137 located using this lookup specification.
138 
139 PrintCenter informs the printer browser module of the number of lookup
140 specification dictionaries that are available when the module's Initialize()
141 function is called. Thereafter, the printer browser module can obtain
142 a copy of any of the lookup specification dictionaries by calling the
143 PMPrBrowserGetLookupSpec callback and specifying an index between 0 and n-1,
144 where n is the number of lookup specification dictionaries.
145 
146 PrintCenter creates a full copy of the printer lookup specification dictionary
147 before passing it to the printer browser module; the printer browser module
148 should call ::CFRelease() on the dictionary when it is no longer needed.
149 */
150 #define kPMPrBrowserLookupRefStr "PrBrowser Lookup Reference"
151 #define kPMPrBrowserLookupRefKey CFSTR("PrBrowser Lookup Reference")
152 /*
153 Keys for the printer selection specification dictionary. This CFDictionary
154 specifies the printers selected in the browser view. An array of these
155 dictionaries is is passed back to PrintCenter when it calls the printer
156 browser module's GetSelectedPrinters() function.
157 
158 An array of selection specification dictionaries is passed from PrintCenter
159 to the printer browser module when PrintCenter calls the WorksetPrinters()
160 function. These dictionaries specify which printers are already in the
161 Workset. The printer browser module can use this information to indicate in
162 its UI whether a printer has been added to the Workset. It can also use
163 this information in non-UI mode to avoid supplying duplicate printers during
164 automatic printer discovery.
165 
166 kPMPrBrowserSelectNameKey corresponds to a CFStringRef to a human-readable
167 printer name string, obtained from the printer or otherwise constructed by
168 the printer browser module.
169 
170 kPMPrBrowserSelectKindKey corresponds to a CFStringRef to a human-readable
171 printer description string that can be displayed in the PrintCenter Workset
172 window. For printer that uses Apple's PostScript printer module over an
173 AppleTalk connection, this would be the string "PostScript printer". Often
174 this is the same as the string corresponding to kPMPrBrowserLookupKindKey
175 in the printer lookup specification dictionary, but it is possible that
176 more specific printer model information might be obtained by the printer
177 browser module during the lookup.
178 
179 kPMPrBrowserSelectAddrKey corresponds to a CFDataRef to a connection-specific
180 address specification for the printer. On an AppleTalk connection, this would
181 be a standard NBP address of the form <name>:<type>@<zone>.
182 
183 kPMPrBrowserSelectRefKey corresponds to a CFTypeRef whose value is the same
184 as that corresponding to kPMPrBrowserLookupRefKey in the printer lookup
185 specification dictionary. The reference value in the lookup specification
186 dictionary that was used to locate the selected printer must be copied into
187 the printer selection specification dictionary and assigned this key.
188 
189 When PrintCenter supplies printer selection specification dictionaries using
190 the WorksetPrinters() function, the dictionaries do not contain a
191 kPMPrBrowserSelectRefKey entry; the printer browser module should not expect
192 to find a corresponding value for this key in those dictionaries.
193 
194 PrintCenter gets information about selected printers when it calls the
195 printer browser module's GetSelectedPrinters() function.
196 
197 PrintCenter will call ::CFRelease() on the printer selection specification
198 dictionary when it is no longer needed.
199 
200 The printer browser module should call ::CFRelease() on the array of
201 printer selection specification dictionaries supplied via the WorksetPrinters()
202 function when they are no longer needed.
203 */
204 #define kPMPrBrowserSelectNameStr "PrBrowser Selection Name"
205 #define kPMPrBrowserSelectNameKey CFSTR("PrBrowser Selection Name")
206 #define kPMPrBrowserSelectKindStr "PrBrowser Selection Kind"
207 #define kPMPrBrowserSelectKindKey CFSTR("PrBrowser Selection Kind")
208 #define kPMPrBrowserSelectAddrStr "PrBrowser Selection Address"
209 #define kPMPrBrowserSelectAddrKey CFSTR("PrBrowser Selection Address")
210 #define kPMPrBrowserSelectRefStr "PrBrowser Selection Reference"
211 #define kPMPrBrowserSelectRefKey CFSTR("PrBrowser Selection Reference")
