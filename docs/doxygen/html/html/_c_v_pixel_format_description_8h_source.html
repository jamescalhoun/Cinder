53 /* This works in close conjunction with BitsPerBlock */
54 /* Examples:
55  8-bit luminance only, BlockWidth would be 1, BitsPerBlock would be 8
56  16-bit 1555 RGB, BlockWidth would be 1, BitsPerBlock would be 16
57  32-bit 8888 ARGB, BlockWidth would be 1, BitsPerBlock would be 32
58  2vuy (CbYCrY), BlockWidth would be 2, BitsPerBlock would be 32
59  1-bit bitmap, BlockWidth would be 8, BitsPerBlock would be 8
60  v210, BlockWidth would be 6, BitsPerBlock would be 128 */
