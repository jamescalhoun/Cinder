122 /* Control flags for TruncateForUnicodeToText: */
123 /*
124  Now TruncateForUnicodeToText uses control flags from the same set as used by
125  ConvertFromTextToUnicode, ConvertFromUnicodeToText, etc., but only
126  kUnicodeStringUnterminatedMask is meaningful for TruncateForUnicodeToText.
127 
128  Previously two special control flags were defined for TruncateForUnicodeToText:
129  kUnicodeTextElementSafeBit = 0
130  kUnicodeRestartSafeBit = 1
131  However, neither of these was implemented.
132  Instead of implementing kUnicodeTextElementSafeBit, we now use
133  kUnicodeStringUnterminatedMask since it accomplishes the same thing and avoids
134  having special flags just for TruncateForUnicodeToText
135  Also, kUnicodeRestartSafeBit is unnecessary, since restart-safeness is handled by
136  setting kUnicodeKeepInfoBit with ConvertFromUnicodeToText.
137  If TruncateForUnicodeToText is called with one or both of the old special control
138  flags set (bits 0 or 1), it will not generate a paramErr, but the old bits have no
139  effect on its operation.
140 */
